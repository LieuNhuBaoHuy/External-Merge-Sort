Truy cập: https://github.com/LieuNhuBaoHuy/External-Merge-Sort
Bấm nút Code màu xanh lá và chọn Download ZIP
Sau khi file .zip được tải về, extract nó.
Trong file đã được extract, tìm mở file cấu hình “external_merge_sort_demo.sln” trong directory: " External-Merge-Sort-main/External-Merge-Sort-main/external_merge_sort_demo".
Trong visual studio, bấm chạy sẽ mở ra một trang web bằng browser.
Nếu có sẵn file cần được sort, bấm vào ô chữ nhật nét đứt để tìm chọn file đó. 
Sau khi tìm chọn xong thì bấm nút sort để download kết quả.
Nếu không có sẵn file và cần tạo một file ngẫu nhiên nào đó để thử nghiệm, bấm dòng chữ get a testing file.
Để đọc kiểm tra một file, dịch chuyển file vào directory: " External-Merge-Sort-main/External-Merge-Sort-main/temp/temp ". bên trong thư mục đã extract nói trên. 
Tìm mở file cấu hình “temp.sln” và bấm chạy.
Trong terminal của chương trình đang chạy, nhập đầu đuôi tên file cần đọc và enter.
Chương trình cũng đã cung cấp sẵn hai file thử nghiệm có tên “sorted.bin” và “sorted_sample.bin” mà ta có thử đọc trong terminal.
